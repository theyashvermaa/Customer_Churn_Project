# Telecom Churn Prediction System

## Overview
This project provides an end-to-end solution for predicting telecom customer churn. It includes a data analysis pipeline and a Streamlit web application for real-time inference.

## Setup Instructions
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `streamlit run app.py`

## Project Components
- `app.py`: Streamlit frontend.
- `best_model.pkl`: Serialized Scikit-Learn pipeline.
- `notebook.ipynb`: Training and EDA workflow.
